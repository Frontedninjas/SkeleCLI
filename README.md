# skelecli

A simple CLI to generate a Template  for a new project

## Usage

```bash
skelecli
```

## Installation

```bash
npx skelecli
```
## global installation 

```bash
npm install -g skelecli
```

## Features
- react+Tailwind  template
- react+Tailwindcss+Typescript template
- Astro+Tailwindcss+Typescript template
